Topic Question: What is the difference between Margin, Border, and Padding?



The element is built up of four different components. The Content, Padding, Border, Margin. The border is in between the padding and margin. If you increase the padding size you are pushing the border away from the content. If you increase the margin size you are moving the element further into the page. You use margin when you want to control the space outside the element/border and you use padding when you want to control the space inside of the element/border. You can control margin by using it like a clock. The first number would be margin: 40px which would push the element away from the top page by 40 pixels. the next number would be right, bottom and then left. Example margin: 40px, 40px, 88px, 72px.
